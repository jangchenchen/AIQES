电梯主要安全保护装置的检查、调整、复位和校验要求，以**【部件｜大白话版本释义】**的格式整理如下表格，以便您更清晰地理解其核心要点：

| 部件 | 大白话版本释义（检查、调整、复位、更换或试验要求） |
| :--- | :--- |
| **称重装置** | **检查要点：** 确认称重传感器或开关的机械连接牢固 [1a]。必须确认电梯在满载和超载时的信号准确无误 [1b]。当电梯超载（110%额定载荷）时，必须发出清晰的声光警报，并且电梯门必须保持完全开启，阻止运行 [1c]。 **调整方法：** 使用100%额定载荷来调整满载开关（如有），确保超载开关此时不动作 [1a]。使用110%额定载荷时，调整超载开关的位置使其动作 [1b]。同时，检查并紧固所有机械部件的螺栓 [1c]。 **校验周期：** 每年至少进行一次功能检查。 |
| **限速器** | **检查要点：** 调节螺钉上的铅封或封记必须完整，不能有被破坏的痕迹 [2a]。限速器运转时必须平稳顺畅，不能有任何摩擦、卡住或转动不灵活的现象 [2b]。所有电气开关和机械零件必须功能正常 [2c]。需要对轴承和离心甩动装置进行润滑 [2d]，并检查钢丝绳的磨损情况 [2e]。 **动作后复位：** 让电梯以检修速度向相反方向微动运行 [2a]。手动将限速器内部的机械锁止件（制动棘爪）恢复到原始位置 [2b]。重新接通限速器上的电气安全开关（即复位开关）[2c]。 **校验周期：** 如果限速器使用年限不超过15年，需要每**2年**进行一次动作速度校验 [2a]。如果使用年限超过15年，则需要**每年**进行一次动作速度校验 [2b]。 |
| **限速器张紧装置** | **检查要点：** 检查张紧装置的机械结构是否完整、没有变形或损坏 [3a]，并确保其提供的张紧力正常 [3a]。检查张紧轮的轴承是否有损坏 [3b]。检查张紧力不足时应动作的“张紧检查开关”是否完好且功能有效 [3c]。同时，检查张紧轮的绳槽是否有缺损或严重磨损 [3a]。 **调整内容：** 调整张紧开关的位置，确保它能在张紧力不够时有效断开安全回路 [3a]。调整张紧轮距离地面或其他部件的最小安全间隙 [3b]。 **更换要求：** 如果绳槽出现缺损或严重磨损，或者张紧装置的机械结构发生变形、损坏，以及张紧检查开关功能失效或破损时，必须进行更换。 |
| **安全钳** | **清理和紧固：** 彻底清除安全钳所有部件上的灰尘、污垢和旧润滑材料 [4a]。取出并清理制动块（闸瓦和楔块）的工作面，涂抹厂家规定的专用刹车油后重新装回 [4b]。检查安全钳动作时的行程距离是否在允许的范围内 [4c]。拧紧安全钳底座的固定螺栓，并确认底座上的可靠标记没有发生位移 [4d]。 **间隙调整：** 调整水平和垂直拉杆上的连接接头，目的是使安全钳的**每个楔块与导轨的工作面之间保持 $2 \sim 3mm$ 的均匀间隙**。调整完成后，固定接头的位置。 |
| **夹绳器** | **检查要点：** 检查夹绳器结构是否完整，所有配件是否齐全 [5a]。确保运动和摩擦部位没有油污堆积 [5b]。检查夹绳器的摩擦片状况 [5c]。最关键的是，检查夹绳器的动作可靠性，确保它能灵活可靠地制停以额定速度上行的**空载轿厢** [5d]。 **手动复位：** 拧动复位螺杆 [5a]。使用复位扳手将夹绳机构和摩擦片恢复到正常待命位置 [5b]。复位电气安全开关 [5c]。然后，点动电梯先向下，再向上运行，以确认夹绳器完全归位，且运行过程中没有发生刮擦 [5d]。 **更换要求：** 夹绳器摩擦片的磨损程度超过了制造厂家规定的设计极限值，或者达到了规定的工作次数时，就必须更换。 |
| **缓冲器** | **检查要点：** 检查缓冲器安装是否牢固，外观是否完好 [6a]。对于耗能型（油压）缓冲器，必须检查液位是否正常 [6b]。检查缓冲器上的电气安全开关是否完好 [6c]。确认对重块下方的撞板与缓冲器顶面的垂直距离没有超过越程距离标识 [6d]。 **报废标准：** 缓冲器动作后，如果出现影响正常工作的永久变形、断裂、塑性变形，或耗能型缓冲器的柱塞严重锈蚀导致无法完全复位，则应报废 [6a, 6b, 6c]。 **选型依据：** 蓄能型缓冲器仅适用于额定速度**小于或等于 $1m/s$** 的电梯 [6a]。耗能型缓冲器可用于**任何额定速度**的电梯 [6b]。 |
| **限速器与安全钳联动试验与调整** | **试验载荷：** 试验前，必须先确认平衡系数试验合格。根据安全钳类型确定载荷：瞬时式安全钳使用**额定载重量** [7a]；渐进式安全钳使用**125%额定载重量** [7b]。 **试验步骤：** 装载规定载荷后，以检修速度向下运行 [7a]。首先，人为分别触发限速器和安全钳的电气开关，确认电梯停运 [7b]。然后，短接这些电气开关，以检修速度下行时，人为触发限速器，观察轿厢机械制停情况 [7c, 7d]。 **复位步骤：** 慢速上行 [7e]。手动复位限速器的机械装置 [7f]。复位限速器和安全钳上的所有电气安全开关 [7g]。最后，检查并确认安全钳的楔块已经完全松开归位 [7h]。 **轿底斜度要求及调整：** 轿厢被安全钳制停后，轿厢地板的倾斜度**不应大于5%** [7a]。调整方法是使用塞尺精确测量并调整安全钳模块与导轨两侧工作面之间的间隙，确保间隙均匀且符合厂家规定的数值 [7b]。 |
